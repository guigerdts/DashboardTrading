# Analytics Module

## Purpose

Provides quantitative analysis of trading performance — metrics, charts,
reports, and statistical tools.

## Responsibilities

- Compute performance metrics (Sharpe, Sortino, max drawdown, etc.)
- Generate equity curve and trade distribution visualisations
- Run period-over-period comparisons
- Future: ML-driven pattern recognition

## Allowed Dependencies

- `app.core` (config, database, dependencies)

## Status

**Scaffold** — base structure, pending implementation.
