# Strategies Module

## Purpose

Manages trading strategy definitions, backtest configurations, and
forward-test tracking.

## Responsibilities

- Define and store strategy parameters and rules
- Trigger backtest runs and store results
- Track forward-test performance vs. backtest projections
- Future: ML model registration and lifecycle

## Allowed Dependencies

- `app.core` (config, database, dependencies)
