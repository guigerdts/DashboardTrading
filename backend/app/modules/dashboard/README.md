# Dashboard Module

## Purpose

Provides the main dashboard view for the Trade Intelligence Platform — an
at-a-glance summary of trading performance, key metrics, and recent activity.

## Responsibilities

- Aggregate and display key trading metrics (PNL, win rate, drawdown)
- Surface recent trades, journal entries, and active errors
- Provide quick-access navigation to other modules
- Future: customisable widget layout

## Allowed Dependencies

- `app.core` (config, database, dependencies)

## Status

**Scaffold** — base structure, pending implementation.
