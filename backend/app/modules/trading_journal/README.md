# Trading Journal Module

## Purpose

Records, displays, and manages trade journal entries — the core data source
for post-trade analysis and review.

## Responsibilities

- Create, read, update, and delete journal entries
- Tag and categorise trades for later filtering
- Attach screenshots and notes to entries
- Provide search and export functionality

## Allowed Dependencies

- `app.core` (config, database, dependencies)

## Status

**Scaffold** — base structure, pending implementation.
