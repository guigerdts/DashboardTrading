# Screenshot Library Module

## Purpose

Manages the storage, annotation, and retrieval of trading screenshots —
charts, trade confirmations, and other visual records.

## Responsibilities

- Upload and store screenshots (local file system or cloud)
- Annotate and tag screenshots for later search
- Link screenshots to journal entries and setups
- Provide thumbnail gallery and full-view display

## Allowed Dependencies

- `app.core` (config, database, dependencies)
