# Psychology Module

## Purpose

Tracks and analyses the psychological aspects of trading — emotional state,
decision patterns, and behavioural biases.

## Responsibilities

- Log emotional state before/after each trade
- Identify recurring behavioural patterns (FOMO, revenge trading, etc.)
- Generate psychology reports and streak tracking
- Provide reflection prompts and exercises

## Allowed Dependencies

- `app.core` (config, database, dependencies)
