# Setups Module

## Purpose

Catalogues and manages trade setups — recurring chart patterns, entry
conditions, and configuration templates.

## Responsibilities

- Define setup templates with entry/exit conditions
- Tag trades with the setup they belong to
- Track setup performance and reliability statistics
- Provide setup discovery and filtering

## Allowed Dependencies

- `app.core` (config, database, dependencies)

## Status

**Scaffold** — base structure, pending implementation.
